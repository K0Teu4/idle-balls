# Idle Balls

Idle Plinko on **Phaser 4** + **Matter.js** + **Vite**.

## Run

```bash
npm install
npm run dev
```

## Controls

| Key / UI | Action |
|----------|--------|
| **Space** | Drop ball |
| **Esc** | Close overlays |
| **Daily** | Claim daily streak bonus |
| Shop wheel | Scroll upgrades |

Save: `localStorage` → `idle-balls-save` (migrates to v6).
